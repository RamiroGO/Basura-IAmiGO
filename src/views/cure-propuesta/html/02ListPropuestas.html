Descripción General

Se presentarán propuestas para la gestión de servicios de la base de datos médica

Especificaciones

  1. Implementar tecnologías gratuitas.
  2. Escalabilidad en la capacidad de Gestionar una Base de Datos.
  3. Facilidad para la edición de la información.
  4. Capacidad de implementar tecnologías que hagan uso tanto de paradigmas de programación imperativos como de los funcionales.
  5. Compatible con tecnologías que hagan uso del lenguaje R.
  6. Discutir sobre la implementación de un Servidor básico para regular la información desde el cliente con la base de datos.
  7. Analizar la manera en la que se viene estructurando la información regularmente y discutir posibles mejoras.
  8. Implementar servicios según sean planteados en las propuestas aceptadas.
